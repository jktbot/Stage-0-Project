Key Points at Stage 0

Web Page

Web pages are text documents written in HTML.
Browsers read these documents, interpret them, and then display them.

How Coding Works

Coding is done by programmers who tell computers what to do in specialized languages that computers can understand. Codes give computers intructions that computers can follow and execute.

Computers are Stupid

Computers must be instructed exactly the way computers can understand, or else the program would not work. Programmers must follow specific sets of syntax .

Programmers Can't Remember Everything

There's just too many concepts to learn so it is okay for programmers to look up things they don't know.

Basic HTML Vocabulary

Tags - Tags are always contained within angled brackets < >. Most tags have an opening tag and a closing tag, but some tags don`t have a closing tag. Tags that don`t have a closing tag are called "void tags."

Element - HTML elements refer to everything within a set of opening and closing tags.

Attribute - An attribute is a property of an HTML element.
